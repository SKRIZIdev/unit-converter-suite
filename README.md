# 🔁 Unit Converter Suite (Convertly)

A six-category unit converter — length, weight, volume, speed, digital storage and temperature — with instant two-way conversion and a swap button. Includes a quick-reference sheet of common conversions.

**Live demo:** https://skrizidev.github.io/unit-converter-suite/

## Pages
- **Convert** — tabbed converter with live results
- **Reference** — common conversions at a glance
- **About** — how the math works (base-unit method + temperature special-casing)

## Tech
Vanilla HTML / CSS / JavaScript. No dependencies.
